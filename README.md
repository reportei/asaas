# Reportei Asaas SDK

Instalação
----------

Adicione os seguintes códigos no seu `composer.json`:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/reportei/asaas"
        }
    ],
    "require": {
        "reportei/asaas": "dev-main"
    }
}
```

Rode os seguintes comandos:

```bash
composer require reportei/asaas
```

```bash
composer dump-autoload
```
